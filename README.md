# Marketing_Account_Bulider
#营销号生成器

**基于Spring MVC开发的Web端营销号生成器**
实现原理十分简单

已打包好可以直接放tomcat服务器上跑的webapps包，见仓库/creating/，丢到tomcat/webapps包
下即可直接通过
ip:port/creating访问页面
效果浏览：
http://sdx.bswaterb.club/creating/

项目源码见仓库/untitled2/
